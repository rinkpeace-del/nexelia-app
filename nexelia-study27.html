There is / There are とは？
「〜がある・いる」を表す文の形
後ろに続く名詞が 1つ（単数） のとき: There is
後ろに続く名詞が 2つ以上（複数） のとき: There are
形
使い方
There is ...
後ろの名詞が1つ（単数）のとき
There are ...
後ろの名詞が2つ以上（複数）のとき
ポイント: 「何があるか・いるか」を言うときに使います。s が付く複数の名詞なら are に注目。
There is / There areとは？・スライド 2
例文で確認
例文で使い分けよう
There is a cat on the desk.
机の上に猫がいます。
There are three books on the desk.
机の上に本が3冊あります。
There is a park near my house.
私の家の近くに公園があります。
There is: 後ろが単数（a cat / a park）
There are: 後ろが複数（three books）
There is / There areとは？・まとめ
このレッスンで学んだこと
まとめ
1
There is / There are は「〜がある・いる」を表す。
2
単数なら There is、複数なら There are。
3
is / are は 後ろの名詞の数 で決まる。
3つのポイントを押さえたら演習に進もう。
