仮定法過去の公式
📐 仮定法過去の形
If + S + 動詞の過去形
「もし〜なら」の部分
,
S + would / could + 動詞の原形
「〜するのに」の部分
🔁 Lesson 1の復習：なぜ過去形？
「今の現実から遠ざかる」ために時制を1つずらすから。過去形 ＝ 非現実のサインだよ！
would
〜するだろうに（意志・予想）
I would buy it.
（買うだろうに）
could
〜できるのに（可能性・能力）
I could go there.
（行けるのに／行ける可能性があるのに）
💡 核となる判断軸：If + 過去形, would / could + 原形
would ＝ するだろうに　／　could ＝ できるのに・可能性があるのに
Unit 19 · レッスン2 · スライド 2
このスライドで学ぶこと
動詞をどう変える？
現在形（元の形）
仮定法（過去形）
am / is / are
→
were（全員！）
have / has
→
had　例：If I had more time,
know / knows
→
knew　例：If I knew her name,
can → could
⚑
後半（〜できるのに）専用
例：… I could join the team.
will → would
⚑
後半（〜するだろうに）専用
例：… I would travel the world.
⚠️ よくあるミス2つ
❌ be動詞を was にしてしまう → 正しくは were（主語にかかわらず！）
❌ 後半の動詞を will / can のままにしてしまう → would / could に変える！
💡 be動詞は I / he / she / they … 全員 were！
これだけは絶対に覚えよう。
Unit 19 · レッスン2 · スライド 3
このスライドで学ぶこと
肯定文・否定文・疑問文
【肯定文】
If I were you, I would apologize.
もし私があなたなら、謝るのに。
【否定文】— would not / could not
If I had a car, I wouldn't take the bus.
もし車があれば、バスに乗らないのに。
would not → wouldn't と短縮できるよ
【疑問文】— Would + 主語 … if + 主語 + 過去形 ?
Would you travel abroad if you had more time?
もし時間がもっとあれば、海外旅行に行きますか？
👉 Would を文頭に出して、主語の前に置くよ（疑問文の語順）
👉 If 節は前でも後ろでもOK！
　　前：If you had more time, would you travel abroad?
　　後：Would you travel abroad if you had more time?
💡 否定は wouldn't / couldn't、疑問は Would … if … ?
どの形でも「If のあとは過去形」のルールは変わらないよ！
Unit 19 · レッスン2 · スライド 4
このスライドで学ぶこと
直説法 vs 仮定法を見抜こう
文A — 直説法
If I am free tomorrow, I will join you.
明日ひまなら（本当にそうかもしれない）参加するよ。
文B — 仮定法
If I were free, I would join you.
もしひまだったら（今の現実：ひまじゃない）参加するのに。
🔍 見分け方チェックリスト
① If のあとが 現在形（am / is / know…） → 直説法
② If のあとが 過去形（were / had / knew…） → 仮定法
③ 後半が will / can → 直説法　／　would / could → 仮定法
💡 形を見るだけで判断できる！
If のあとと後半の動詞、2か所だけチェックすればOKだよ。
