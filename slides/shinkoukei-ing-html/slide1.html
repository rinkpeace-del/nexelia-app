ing形の作り方 · スライド 1
このスライドで学ぶこと
基本は +ing
ほとんどの動詞
動詞 + ing
play → playing
eat → eating
read → reading
💡 基本はそのまま +ing をつけるだけ！
ing形の作り方 · スライド 2
このスライドで学ぶこと
e を取って +ing
語尾が e で終わる動詞
e を取って + ing
make → making makeing ではない
use → using useing ではない
write → writing writeing ではない
💡 語尾が e のとき、そのまま +ing にすると読みにくくなるので e を取る！
ing形の作り方 · スライド 3
このスライドで学ぶこと
子音を重ねて +ing
「短母音 + 子音1文字」で終わる動詞
最後の子音を重ねて + ing
run → running n を重ねる
swim → swimming m を重ねる
sit → sitting t を重ねる
💡 run / swim / sit など短くて力強い動詞は子音を重ねることが多い！
ing形の作り方 · まとめ
このレッスンで学んだこと
まとめ
1
基本は +ing（play→playing, eat→eating）
2
語尾が e のとき → e を取って +ing（make→making）
3
語尾が 短母音+子音 のとき → 子音を重ねて +ing（run→running）
💡 この3つが頭に入ったら演習に進もう！
