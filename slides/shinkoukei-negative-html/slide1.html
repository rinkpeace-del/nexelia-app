Unit 3 · レッスン2 · スライド 1
ing形には3つのパターンがある！
1
そのまま ing をつける
play → playing　／　watch → watching
2
e を取って ing をつける
make → making　／　use → using
3
最後の文字を重ねて ing をつける
run → running　／　swim → swimming
💡 どのパターンか迷ったら、動詞の語尾に注目！
次のスライドで1つずつ確認しよう。

Unit 3 · レッスン2 · まとめ
まとめ
3パターンを使い分けよう！
1
ふつうの動詞 → そのまま ing（play→playing）
2
語尾が e → e を取って ing（make→making）
3
「短母音＋子音」で終わる → 最後を重ねて ing（run→running）
⚡ 最短判断ルール
1
語尾が e で終わる？ → YES → e を取って ing
↓ NO
2
「短母音＋子音」で終わる？（例：run, sit）→ YES → 最後を重ねて ing
↓ NO
3
それ以外 → そのまま ing
