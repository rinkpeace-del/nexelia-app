Unit 22 · レッスン4 · スライド 1
時制の総まとめ
現在形 vs 現在進行形
習慣・事実 → 現在形（play / plays）
今まさに動作中 → 現在進行形（am / is / are playing）
know / like など状態動詞 → 進行形にしない
過去形 vs 現在完了形
yesterday / ago → 必ず過去形（現在完了は使えない）
今とのつながりがある → 現在完了形（have / has ＋ 過去分詞）
現在完了の3用法キーワード
経験：ever / never / before
継続：for / since
完了：just / already / yet
💡 迷ったら「いつの話？今と関係ある？」を確認！

準動詞の総まとめ
不定詞（to ＋ 原形）の3用法
名詞的：「〜すること」→ I want to play.
副詞的：「〜するために」→ I study to pass.
形容詞的：「〜するための」→ homework to do
動名詞（ing形）を続ける動詞
enjoy / finish / stop / practice / give up
→ 後ろは基本、動名詞（ing形）
分詞の使い分け
現在分詞（〜ing）＝ 能動「〜している」
過去分詞 ＝ 受動「〜された」
exciting（させる側）vs excited（感じる側）
⚠️ enjoy to play はNG → enjoy playing（ing形）が正しい！

比較・受け身・関係代名詞の総まとめ
比較の3段階
原級：as 〜 as ／ 比較級：〜er than ／ 最上級：the 〜est
長い語 → more / most ／ 最上級には必ず the
more taller / most tallest → 二重比較はNG
受け身（be ＋ 過去分詞）
現在：is / am / are ＋ 過去分詞
過去：was / were ＋ 過去分詞
「〜によって」は by を使う
関係代名詞
先行詞が人 → who ／ もの・動物 → which ／ どちらも → that
主語が足りない → 関係代名詞が主語になる（who lives…）
主語がある → 関係代名詞が目的語になる（who I met…）
💡 この3つは入試頻出！形・語順・theのつけ忘れに注意。
