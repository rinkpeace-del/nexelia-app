意味の確認

問題 3 / 5
because vs so
正しい英文はどれ？
because と so の使い方に注目！
問題 4 / 5
誤文発見
文法的にまちがっている英文はどれ？
if 節の中の動詞の形に注目！
問題 5 / 5
総合判断
日本語に合う英文をすべて選べ。（複数あり）
「もし明日晴れなら、私たちは公園に行きます。」
if 節の動詞の形・接続詞の選択に注目！コンマの有無は問わない。

💡 because ＝「〜だから」／if ＝「もし〜なら」
because と so は同じ文で使わない！どちらか1つだけ
if 節の中は未来でも現在形（will は使わない）
⚠ よくあるミス TOP 3
1. Because …, so … → Because …,（so は不要！）
2. If it will rain → If it rains（if 節の中は現在形！）
3. 文頭の if / because のコンマ忘れ → If …, のカンマを忘れずに
