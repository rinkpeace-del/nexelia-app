💡 もの・動物を説明するときは which（人には who！）
which のうしろは動詞から始める（it などは不要！）
which のうしろの動詞は説明するものに合わせる → 単数・複数に注意
⚠ よくあるミス TOP 3
1. a dog who → a dog which（動物には which！）
2. which it is → which is（which のうしろに it は不要！）
3. books which moves → books which move（books が複数 → s なし！）
