Unit 16 · レッスン2 · スライド 3
which のうしろの動詞の形に注意！
🔍
which のうしろの動詞は「説明するもの」に合わせる
→ which ＝ そのもの（主語） の役割を持っているから
The train which leaves at 8 is fast.
8時に出発する電車は速い。
→ the train（3人称単数）→ leaves（s あり）
I like songs which have a good melody.
私はメロディーの良い歌が好きだ。
→ songs（複数）→ have（s なし）
⚠️ which のうしろも、説明するものに合わせて動詞の形が変わる！
単数なら s あり、複数なら s なし。
Unit 16 · レッスン2 · スライド 4
よくあるミスに気をつけよう！
❌
I have a dog who runs fast.
✅
I have a dog which runs fast.
→ dog はもの・動物 → which を使う（who は人専用！）
❌
This is the book which it is interesting.
✅
This is the book which is interesting.
→ which のうしろに it を重ねない！which がすでに「そのもの（主語）」の役割を持っている。
⚠️ who と which の混同・うしろへの代名詞の重ねは2大ミス！
