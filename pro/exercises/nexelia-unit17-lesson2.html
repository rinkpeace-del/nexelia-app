Unit 17 · レッスン2 · スライド 1
because ＝「〜だから・〜なので」
because
接続詞（理由）
「〜だから」「〜なので」
because のあとに「理由（主語＋動詞）」を続ける
I was tired because I ran a lot.
たくさん走ったので、私は疲れた。
Because it was cold, I wore a coat.
寒かったので、私はコートを着た。
💡 because も文頭に置ける！
文頭に置くとき → 後ろにコンマ（,）が必要（L1 と同じルール）
Unit 17 · レッスン2 · スライド 2
because と so は同じ文で使わない！
✗
Because I was tired, so I went to bed.
because と so を両方使うのは誤り
✓
Because I was tired, I went to bed.
because だけ使う
✓
I was tired, so I went to bed.
so だけ使う
⚠️ because ＝「〜だから」（理由が後ろ）
so ＝「だから〜」（結果が後ろ）
どちらか1つだけ使う！両方は NG！
Unit 17 · レッスン2 · スライド 3
if ＝「もし〜なら」
if
接続詞（条件）
「もし〜なら」「〜の場合は」
if のあとに「条件（主語＋動詞）」を続ける
If it rains tomorrow, I will stay home.
もし明日雨が降ったら、私は家にいます。
I will call you if I have time.
時間があれば、電話します。
💡 if も文頭に置ける！
文頭に置くとき → 後ろにコンマ（,）（L1 と同じルール）
Unit 17 · レッスン2 · スライド 4
if 節の中は未来でも現在形！
✗
If it will rain tomorrow, I will stay home.
if 節の中に will は使わない
✓
If it rains tomorrow, I will stay home.
if 節の中は現在形（rains）
⚠️ 「もし明日〜なら」と未来のことを言うときも、
if のあとは will を使わず現在形にする！
will は if の外（主節）に置く。
💡 まとめると：
If ＋ 現在形, will ＋ 動詞の原形 が基本の形！
