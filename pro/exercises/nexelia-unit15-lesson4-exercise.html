💡 動名詞 総まとめ
① 形：動詞 ＋ ing（e省略 / 子音重ねに注意）
② 働き：主語・目的語・前置詞の後（必ず ing）
③ enjoy / finish / mind → ing のみ
④ like / love / start → ing でも to でも OK
⚠ よくあるミス TOP 3
1. run → runing → running（子音を重ねる！）
2. enjoyed to play → enjoyed playing（enjoy は ing のみ！）
3. Cooking are → Cooking is（動名詞主語は単数！）
