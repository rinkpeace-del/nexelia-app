問題 5 / 5
誤文発見
文法的にまちがっている英文はどれ？
If 節の形・would のあとの動詞・were の使い方——全部チェック！

💡 仮定法 ＝ 現実とは違う想像（過去の話ではない！）
If のあとは過去形（be動詞は主語に関わらず were）
主節は would ＋ 動詞の原形のセット
⚠ よくあるミス TOP 3
1. If I am → If I were（仮定法の be動詞は were！）
2. I would went → I would go（would のあとは原形！）
3. If I will have → If I had（If 節に will は使わない）
