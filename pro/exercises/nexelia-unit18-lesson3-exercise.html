問題 3 / 5

I don't know if she （　　　） soccer.
元の疑問文：Does she play soccer?
問題 4 / 5
誤文発見
文法的にまちがっている英文はどれ？
if の使い方と語順に注目！
問題 5 / 5
総合判断
日本語に合う正しい英文はどれ？
「あなたは彼女が駅をどこで降りるか知っていますか？」
疑問詞が必要？ if が必要？ — まず元の疑問文を考えよう！

💡 疑問詞あり → 疑問詞をそのまま使う
Yes/No 疑問文 → if（または whether） を使う
if / whether のあとも 主語＋動詞 の順（疑問文の語順にしない！）
⚠ よくあるミス TOP 3
1. if is he at home → if he is at home（if のあとも主語→動詞！）
2. if she does play → if she plays（does は消えて動詞に s をつける）
3. 「どこで」なのに if を使う → where を使う（疑問詞があるときは if 不要）
