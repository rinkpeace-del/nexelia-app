Unit 17 · レッスン4 · スライド 2
テストで狙われるよくあるミス
✗
Because I was tired, so I slept.
because と so の両方使いは NG → どちらか1つだけ
✗
If it will rain tomorrow, I will stay home.
if 節の中に will は不要 → 現在形（rains）にする
✗
She said that passed she the exam.
that の後ろは「主語 ＋ 動詞」の順 → that she passed が正しい
⚠️ この3つがテストで最も狙われるポイント！
文頭コンマのルール（when / before / after / because / if）も忘れずに。
Unit 17 · レッスン4 · スライド 3
接続詞の判断フロー
⚡ 接続詞を選ぶ手順
1
「いつのこと？」→ when / before / after
↓ 違う場合
2
「なぜ？理由を言う？」→ because（so は使わない！）
↓ 違う場合
3
「もし〜なら？条件？」→ if（節の中は現在形！）
↓ 違う場合
4
「〜と思う・知る・言う？（think / know / say など）」→ that（省略もOK）
💡 迷ったら日本語の意味から逆算して選ぶのがコツ！
文頭に来る接続詞節（that 以外）→ コンマ（,）を忘れずに
Unit 17 · レッスン4 · スライド 4
if と when の使い分け
💡
If it rains tomorrow, I will stay home.
if ＝「もし雨なら」— 雨が降るかどうか不確か（条件）
💡
When it rains, I always stay home.
when ＝「雨のとき（一般・習慣・確実なこと）」— 未来に確実に起きることにも使う（時）
⚠️ if ＝ 起きるかどうかわからない（条件）
when ＝ 起きたとき・起きるとき（時）
「もし」がつく日本語 → if　「〜するとき」の日本語 → when
💡 どちらも節の中は現在形！
文頭に来るとき → コンマ（,）を忘れずに。
