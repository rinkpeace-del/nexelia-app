Unit 17 · レッスン2 · スライド 1
このスライドで学ぶこと
もの・動物には which を使う
who
「人」を説明するとき a teacher who knows a lot（たくさん知っている先生）
which
「もの・動物」を説明するとき a book which is very famous（とても有名な本）
💡 which の代わりに that もよく使われるよ。詳しくは次のレッスンで！
前のレッスンで学んだ who は「人」専用だったね。
「もの・動物」を説明したいときは which（または that）を使おう！
Unit 17 · レッスン2 · スライド 2

which は「it / they」の代わりに使う
もの・動物の名詞
＋
which
＋
動詞 ...
今回は which が主語になるパターン（which ＋ 動詞）
It → which which は「そのもの（it / they）」の代わり！
💡 which は it や they の代わりに使う言葉だよ！
who と同じで、which の後ろも必ず動詞が続くよ。
Unit 17 · レッスン2 · スライド 3
このスライドで学ぶこと
2つの文を which でつなごう
a bag which has many pockets
たくさんのポケットがあるかばん　（もの）
a dog which runs very fast
とても速く走る犬　（動物）
文①
I have a cat.
文②
It → which runs very fast.
↓
完成
I have a cat which runs very fast.
私はとても速く走る猫を飼っています。
✗ NG
I have a cat who runs very fast.
猫は「人」ではないので who は使えない！
Unit 17 · レッスン2 · まとめ
このレッスンで学んだこと
まとめ
1
which は先行詞が「もの・動物」のときに使う
2
which は「it / they」の代わり。後ろに動詞が続く（主格）
3
which の代わりに that もよく使われる
人 → who
もの・動物 → which（または that）
💡 おつかれさま！次は実際に which を使って文をつなぐ問題に挑戦しよう！
