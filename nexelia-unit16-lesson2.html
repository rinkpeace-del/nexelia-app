過去分詞とは何か？
動詞の過去分詞形
break → broken / use → used
＝
「〜された」「〜されている状態の」
名詞を説明する形容詞として使う
break
→
broken
壊されている状態の
不規則
use
→
used
使われた状態の
規則
write
→
written
書かれた状態の
不規則
💡 過去分詞は 「〜されている状態の」 という意味で名詞を説明する形容詞として使えるよ！
現在分詞（〜している）とは意味が逆になるから注意しよう。

このスライドで学ぶこと
過去分詞で名詞を説明する
a broken window
壊されている窓（1語 → 名詞の前）
a used car
使われた車＝中古車（1語 → 名詞の前）
the letter written by him
彼によって書かれた手紙（2語以上 → 名詞の後ろ）
the car made in Japan
日本で作られた車（2語以上 → 名詞の後ろ）
📌 前 or 後ろ？ 判断ルール（Lesson 1 と同じ！）
1語
→
名詞の前
　a broken window
2語以上
→
名詞の後ろ
　the letter written by him
💬 長くなるから後ろに置く — それだけ！
Unit 16 · レッスン2 · スライド 3
このスライドで学ぶこと
現在分詞 vs 過去分詞の意味の違い
現在分詞（Lesson 1）
主語が動く側
〜している
a running boy
走っている少年（能動）
過去分詞（Lesson 2）
主語が受ける側
〜されている
a broken window
壊されている窓（受け身）
the boy writing a letter
手紙を書いている少年　← 少年が「書く」側
the letter written by him
彼によって書かれた手紙　← 手紙が「書かれる」側
💡 選ぶコツは「動く側か？受ける側か？」で考えること！
〜している（動く）→ 現在分詞　／　〜されている（受ける）→ 過去分詞
