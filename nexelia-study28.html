There is / There are の否定文 ・ スライド 1
このスライドで学ぶこと
否定文の作り方
There is / There are の否定文は、
is / are の後ろに not を置くだけ。
There is → There is not / There isn't
There are → There are not / There aren't
ポイント: is / are の後ろに not を入れる。短縮形は isn't / aren't。
There is / There are の否定文 ・ スライド 2
例文で確認
例文で練習しよう
There isn't a cat on the desk.
机の上に猫はいません。
There aren't any books on the desk.
机の上に本は1冊もありません。
There isn't a park near my house.
私の家の近くに公園はありません。
否定文では any を使うことも多い。There aren't any books. など。
