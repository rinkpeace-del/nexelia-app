Unit 21 · レッスン3 · スライド 1
このスライドで学ぶこと
Yes/No 疑問文を
文の中に入れるときは if ！
疑問詞あり
（what / where…）
→ 疑問詞 をそのまま使う
I know where he is. / Do you know what she wants?
Yes/No 疑問文
（Is she ...? など）
→ if または whether を使う
I don't know if she is busy. / Do you know if he came?
💡 判断軸：疑問詞がある？→ 疑問詞をそのまま使う
　　　　　疑問詞がない？→ if（または whether）を使う
Unit 21 · レッスン3 · スライド 2
このスライドで学ぶこと
変換してみよう
if ＋ 主語 ＋ 動詞
🔄 Yes/No 疑問文 → 間接疑問文（if）
❌
I don't know is she busy.
疑問文の語順のまま＆if が抜けている
↓
✅
I don't know if she is busy.
if ＋ 主語（she）＋ 動詞（is）の順
🔄 過去形の Yes/No 疑問文
❌
Do you know did he come?
did は疑問文用 → 間接疑問文には入れない
↓
✅
Do you know if he came?
if ＋ 主語（he）＋ 動詞（came）。did は不要！
💡 語順ルールは疑問詞のときと同じ！
if ＋ 主語 ＋ 動詞（疑問文の語順にしない・do/did は入れない）
Unit 21 · レッスン3 · スライド 3
このスライドで学ぶこと
if と whetherどう違う？
if
日常会話でよく使う
「〜かどうか」
I don't know if she is busy.
Do you know if he came?
=
whether
やや丁寧・書き言葉的
「〜かどうか」
I don't know whether she is busy.
Do you know whether he came?
⚠️ 中学では if を使えば OK！
whether はほぼ同じ意味で少しフォーマルな表現だよ。
試験では「if または whether」どちらでも正解になることが多いよ。
💡 if も whether も「〜かどうか」で意味は同じ！
まずは if を使いこなせれば十分だよ。
Unit 21 · レッスン3 · まとめ
このレッスンで学んだこと
まとめ
疑問詞がない？→ if ＋ 主語 ＋ 動詞 語順ルールは疑問詞のときと完全共通！
1
Yes/No 疑問文を文の中に入れるときは if（または whether）を使う Is she busy? → I don't know if she is busy.
2
if のあとも 主語 ＋ 動詞 の順（do / did は入れない！） ❌ if did he come　→　✅ if he came
3
if と whether は同じ意味。中学では if で OK！ I don't know if / whether she is busy. どちらも正解
👏 お疲れさま！演習で if の感覚をつかもう！
