命令文とは？
「〜しなさい」「〜してください」を表す文
主語（you）を省略して、動詞の原形 で文を始める。
主語がないのが命令文の特徴です。
動詞の原形 Open / Be...
＋
残り the door / quiet...
Open the door.
ドアを開けなさい。
Be quiet.
静かにしなさい。
ポイント: 主語なしで動詞の原形から始める。be動詞の命令は Be ... の形。
命令文 · スライド 2
このスライドで学ぶこと
Don't ＋ 動詞の原形
「〜してはいけない」「〜しないで」と否定するときは、
文頭に Don't を置いて、あとは動詞の原形を続ける。
否定 Don't
＋
動詞の原形 run / be...
＋
残り here / late...
Don't run here.
ここで走ってはいけません。
Don't be late.
遅刻してはいけません。
ポイント: 否定は Don't + 動詞の原形。be動詞の否定も Don't be ... の形。
命令文 · まとめ
このレッスンで学んだこと
まとめ
1
命令文は主語なしで 動詞の原形 から始める。
2
be動詞の命令文は Be ...（Be quiet. など）。
3
否定の命令文は Don't + 動詞の原形（Don't run. など）。
ポイント: この3つを覚えたら演習へ進もう。
