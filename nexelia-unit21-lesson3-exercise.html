問題 1 / 5 判別
正しい英文はどれ？
💡 Yes/No 疑問文を文の中に入れるときは if が必要！語順にも注意しよう。
問題 2 / 5 語句選択
（　）に入る正しい語はどれ？Do you know （　） he passed the exam?
💡 「彼が試験に合格したかどうか」→ 疑問詞がある？ない？まず確認しよう。
問題 3 / 5 英文選択
日本語に合う英文はどれ？「彼女が昨日学校に来たかどうか知っていますか？」
⚠️ did は疑問文を作る助動詞 → 間接疑問文の中には入れない！過去形（came）をそのまま使おう。
問題 4 / 5 誤文訂正
下線部の間違いを直した文はどれ？I wonder if is he free tomorrow.
⚠️ if のあとも「主語 → 動詞」の順！疑問文の語順（is he）にしてはダメ。
問題 5 / 5 並び替え
日本語に合うように正しく並べた文はどれ？
🎬 シチュエーション
友達に「パーティーがまだやっているかどうか知っている？」と聞きたい。
ヒント：Do you know ＋ if ＋ 主語（the party）＋ 動詞（is）＋ still going on？

📌 このレッスンのポイント
Yes/No 疑問文 → if ＋ 主語 ＋ 動詞
→ if のあとも「主語 → 動詞」の順！
→ do / did は入れない！
→ if ＝ whether（中学では if で OK）
⚠️ よくあるミスと原因
if 忘れ❌ I don't know is she hungry.
　→ ✅ I don't know if she is hungry.
語順ミス❌ if is he free　→　✅ if he is free
did 不要❌ if did she come　→　✅ if she came
